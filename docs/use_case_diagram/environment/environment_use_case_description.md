# 生态环境监测用例图（Ecological Environment Monitoring Use Case Diagram）

该用例图展示了生态环境监测系统中各角色与系统功能的交互关系：

**系统名称：** 国家公园管理系统（National Park Management System）  
**业务模块：** 生态环境监测（Ecological Environment Monitoring）

## 主要参与者及功能

1. **数据分析员（data_analyst）**
   - **Review Environmental Monitoring Data**：查看环境监测数据。
   - **Maintain Indicator Thresholds**：维护指标阈值。
   - **Generate Environmental Analysis Report**：生成环境分析报告。

2. **公园管理员（park_manager）**
   - **View Environmental Monitoring Summary**：查看环境监测总结。
   - **View Abnormal Data Alerts**：查看异常数据警报。

3. **技术员（technician）**
   - **Calibrate Monitoring Devices**：校准监测设备。
   - **Update Device Operational Status**：更新设备运行状态。

4. **系统管理员（system_administrator）**
   - **Manage Monitoring Device Information**：管理监测设备信息。
   - **Configure System Parameters**：配置系统参数。

## 说明

- 各参与者通过系统完成其职责。
- 数据分析员主要负责数据处理与报告生成。
- 公园管理员主要查看数据和警报。
- 技术员和系统管理员负责设备维护和系统管理。
